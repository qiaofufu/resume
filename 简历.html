cill web framework 2021.10-2022.01 单独开发
项目地址: https://github.com/qiaofufu/cill

项目概述

  1. cill是一个基于 Trie 树算法的 Web 后端框架，参考了 Gin 框架的设计，并实现了许多特性
  2. 其中包括支持动态路由的功能，可以非常方便地定义和处理动态路由。此外，还增加了 Context 功能，使得开发人员可以在请求处理过程中存储和共享数据，从而简化代码逻辑。
  3. 此外，提供了分组控制功能，允许将 API 进行分组，以方便管理和维护。同时开发人员还可以通过自定义中间件来扩展请求和响应的处理能力，并灵活地使用这些中间件来满足不同的业务需求。
  4. 本框架还提供了模板功能，我可以使用预定义的模板来生成动态网页，这大大方便了开发工作。同时，错误恢复功能确保了框架在遇到异常情况时也能够正常运行，提高了系统的稳定性。

项目感悟

  • 初步了解了一个web框架应该具备的特性，同时将Trie树 算法应用到路由匹配中，进一步的理解了中间件的扩展能力的重要性
Bankspikekillsystem 2021.10-2022.01 单独开发
项目地址: https://github.com/qiaofufu/bankspikekillsystem

项目概述

  1. 本项目是一个银行理财产品秒杀管理系统，是参加大学生服务外包大赛的作品
  2. 它使用反射数据库表信息来实现列级别的风险决策控制，并使用Redis分布式锁解决超买超卖问题。

  4. 为了应对突发流量，它使用RabbitMQ消息队列进行订单生成异步处理。
  5. 此外，使用Redis缓存技术缓存预订单、库存及热点数据等，并使用云产品如OSS SMS解决数据存储和信息通知问题。
